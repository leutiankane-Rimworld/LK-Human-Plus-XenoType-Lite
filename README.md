# LK-Human-Plus-XenoType-Lite
Adds Evolved Human XenoType , a custom xenotype i made in character editor that i wanted to translate to a mod. This mod takes the original mod i made and rebalances it to be more Vanilla Balanced.  Includes Custom Start Scenarios using the Xeno type basicly crashlanded with reinforcements every 3 days up to 12 colonists in your colony.
